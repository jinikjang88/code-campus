# Code Campus 🎮

신입 개발자를 위한 교육용 2.5D 게임

## 📖 프로젝트 소개

Code Campus는 신입 개발자가 즐겁게 개발 개념을 학습할 수 있도록 설계된 인터랙티브 게임입니다. 
2.5D 탑다운 뷰로 캠퍼스를 탐험하며 다양한 개발 영역(Frontend, Backend, Database)을 경험할 수 있습니다.

### 🎯 주요 특징

- **2.5D 탑다운 뷰**: 깊이감 있는 시각적 경험
- **인터랙티브 학습**: NPC와의 대화를 통한 자연스러운 학습
- **편안한 디자인**: 파스텔 톤의 색상과 부드러운 애니메이션
- **직관적인 조작**: 키보드만으로 간단하게 플레이

## 🚀 시작하기

### 필수 요구사항

- Node.js (v14 이상)
- npm 또는 yarn

### 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/[your-username]/code-campus.git
cd code-campus

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173`으로 접속하세요!

## 🎮 조작법

- **이동**: `W`, `A`, `S`, `D` 또는 방향키
- **상호작용**: `E` (NPC 근처에서)
- **대화 진행**: `Space`

## 🏗️ 기술 스택

- **게임 엔진**: [Kaboom.js](https://kaboomjs.com/) v3000
- **빌드 도구**: Vite
- **언어**: JavaScript (ES6+)
